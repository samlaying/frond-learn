01 - 传统多页 vs SPA

先看入口，再点进去对比

这个目录现在拆成两个独立示例。一个展示传统多页的“整页跳转”，一个展示 SPA 的“局部更新”。

看传统多页 看 SPA

传统多页

点击链接后会跳到另一个 HTML 页面，浏览器重新加载整页内容，地址栏也会跟着变化。

  • 每个页面都是独立文件
  • 跳转时页面整体刷新
  • 适合演示“请求新页面”的感觉

SPA

页面只加载一次，后续点击主要是用 JavaScript 改局部内容，不离开当前页面。

  • 通常只有一个主入口
  • 交互后不整页刷新
  • 适合演示“同一页面内切换视图”

你可以先点“传统多页”，再回到这里点“SPA”，差异会很直观。

前端架构实战入口

这里是总入口。八章共用同一个消息业务，从页面更新一路练到离线综合项目。建议每次只打开一章、只改一个小点，再观察页面日志。

01
SPA 基础

点击按钮只更新局部内容，理解单页应用的核心交互方式。
02
三层架构

把一个功能拆成 UI、业务实体和数据层，练职责分离。
03
单向调用

体会 UI -> 业务层 -> 数据层的单向依赖关系。
04
Event Bus

一个模块只广播消息，多个监听者各自响应。
05
IndexedDB

把结构化草稿保存到浏览器数据库，刷新后仍然存在。
06
Web Worker

对比主线程卡顿与 Worker 消息通信。
07
PWA

注册 Service Worker，让页面外壳可离线打开。
08
离线消息工作台

串联分层、数据库、事件、Worker 与离线缓存。

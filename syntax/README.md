# 前端语法补习

这个目录专门用来补语法。

它不按“完整 JavaScript 教材”来写，而是按当前项目里真实出现过的代码来学。你先在项目里看到一个语法，再回到这里补它的意思。

## 当前主线

- [frontend-architecture-syntax.md](./frontend-architecture-syntax.md)：围绕 `practice/frontend-architecture/` 里出现过的 HTML、CSS、JavaScript 语法做解释。

## 怎么用

建议顺序：

1. 先看 `practice/frontend-architecture/` 里的实际代码。
2. 遇到看不懂的语法，再来这个目录查。
3. 查完立刻回到原代码，看它在那一段里到底解决了什么问题。

不要一开始追求“把语法学全”。当前阶段只要先学会这套练习里已经用到的语法。
